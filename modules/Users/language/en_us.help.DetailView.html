Users Detail Page

View and manage user details such as the user settings and address. If you log in as an administrator, you can duplicate the user record.

  • To edit user information, click Edit.
  • To duplicate user information, click Duplicate. This button displays only when you log in as an administrator.
  • To change the user password, click Change Password, enter the new password, and click Save.
  • To revert back to the default preferences, click the Reset to Default Preferences link at the top right corner of the detail page and click OK to confirm the action
  • If you reconfigured your Home page and want to revert back to the default settings, click the Reset to Default Homepage link at the top right corner of the detail page and click OK to confirm the action.
