Opportunities

Use the Opportunities module to create and manage sales opportunities.

The Opportunities Home page displays the following information:

  • A Search sub-panel where you can enter keywords such as the opportunity name or account name to search for a specific opportunity. To search only for opportunities assigned to you, select Only my items. To perform an advanced search using additional fields, click the Advanced Search tab. To customize and save the search layout and results, click the Saved Search & Layout tab.
  • A Shortcuts section that displays the following options:
    • Create Opportunity. Click this option to create a new opportunity. Alternatively, use the Create Opportunity quick form to enter required information. You can enter additional information after you save it.
    • Opportunities. Click this option to navigate back to the Opportunities Home page from an opportunity's detail page.
    • Opportunity Reports. Click this option to view an existing report on your opportunities, or to create a new report.
    • Import. Click this option to import opportunity information from an external application or file.
  • A Create Opportunity quick form that you can use to create an opportunity with only the required fields. You can enter additional details after you save the opportunity.
  • A list of existing opportunities along with related information such as the account name and amount.
      • To view the opportunity details, click the name or the View icon adjacent to the amount.
      • To edit the opportunity details, click the Edit icon adjacent to the user name; alternatively, click Edit on the detail page.
      • To export one or more opportunities, select them from the list, click the Export link above, and choose Selected Records. To export all the opportunities listed on the page or all the existing opportunities, choose Current Page and Entire List respectively.
      • To update or delete multiple opportunities, select them from the list, and use the Mass Update sub-panel.
