Upgrade Wizard

Use this Wizard to upload and install new Sugar versions or upgrade existing ones. Before you begin this process, ensure that the config.php file for your installation is writable.

  1. Click Next to perform System checks. If the check finds no errors, the status of System Checks changes from Incomplete to Complete.
  2. Click Next to install the upgrade files.
  3. Click Browse to navigate to the location of the upgrade zip file and then click Upload Upgrade; to remove the file, click Delete Package.
    If the upgrade file is loaded successfully, the status of Upload Upgrade changes from Incomplete to Complete.
  4. Click Next to check customized queries.
    To run customized queries, you can select Manual SQL Queries from the Database Update Method drop-down list, and enter your queries in the field that displays below. When the check completes successfully, the status of Preflight Check changes from Incomplete to Complete.
  5. Click Next to begin the upload process.
    When the upgrade is complete, the status of Commit Upgrade changes from Incomplete to Complete.
  6. Click Next to confirm the status of the upgrade.
    When the upgrade is successful, the status of Debrief changes from Incomplete to Complete.
