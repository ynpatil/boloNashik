Diagnostic Tool

Use this page to capture system configuration for diagnostics and analysis. The diagnostic tool pulls up various information about the system and stores it in a zip file on the server for download. When you send a help request to Customer Support, you can attach the file because it summarizes all the basic information required to resolve the problem.

  • To run the diagnostic tool, click Execute Diagnostic.
  • To capture specific configuration information, click the Advanced link below the Execute Diagnostic button. From the list of configuration files, select what you want to view and then click Execute Diagnostic.
    The system creates a Zip file that you can save on your local machine.
