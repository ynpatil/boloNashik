Module Loader

Use this page to upload custom modules, new language packs, and custom themes so that they are included in the current installation.

  1. In the Upload a module field, enter the path to the module or click Browse to navigate to the location of the module .zip file.
  2. Click Upload.
  3. Click Install to unzip the module file; to unload the module, click Delete Package.
  4. Click Commit to install the module.

To uninstall the module, click Uninstall and then click Commit.
