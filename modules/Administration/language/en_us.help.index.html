Administration Home

Use the Administration module to configure and manage Sugar. The Administration Home page displays the following information:

Sugar Network

  • Sugar Support Portal: Click this option to access Sugar Customer Support. You can access the Sugar community forums and the bug tracker, download the application, access the bug tracker, purchase Sugar Network subscriptions, and log into Sugar Network and Sugar University.
  • Sugar Updates: Click this option to access the Configure Sugar Updates page and check for the latest Sugar version. To manually check for updates, click the Check Now button. To instruct the system to periodically check for updates, select the Automatically Check for Updates box.
  • Online Documentation: Click this option to view and download a PDF copies of Sugar Suite user manuals.
  • License Management: Click this option to edit and validate your Sugar license key information.

System

  • System Settings: Click this option to configure the system-wide settings according to your organization's specifications. Users can override some default settings, such as the date format, on their My Accounts page.
  • Scheduler: Click this option to schedule jobs such as dispatching campaign Emails and pruning the database.
  • Diagnostic Tool: Click this option to capture system configuration for diagnostics and analysis.
  • Upgrade Wizard: Click this option to upload and install new Sugar versions as well upgrade existing ones.
  • Locale Settings: Click this option to specify the default system-wide locale settings such as the language, date, and time.
  • Backups: Click this option to backup the Sugar configuration files.
  • Repair: Click this option to upgrade and rebuild data from a previous version of Sugar for the current version of the program. You also use this option to convert a Sugar installation to offline mode.
  • Currencies: Use the Currencies option to define currencies other than the US dollar.
  • Module Loader: Click this option to upload custom modules, new language packs, and custom themes so that they are included in the current installation.

Users

  • User Management: Click this option to create and manage users in Sugar.
  • Vendor Management: Click this option to create and manage vendor in Sugar.
  • Role Management: Click this option to create and manage roles in Sugar.

Email

  • Email Settings: Click this option to set the default settings for outbound emails.
  • Inbound Email: Click this option to monitor and manage emails that your organization receives.
  • Manage Email Queue: Click this option to view, send, and delete mass campaign emails that are in the queue for dispatch.

Studio

  • Studio: Click this option to edit dropdown lists, custom fields, layouts, and lables in Sugar modules.

Bug Tracker

  • Releases: Click this option to maintain a list of your product releases.
