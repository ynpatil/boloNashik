Backups

Use this page to back up the Sugar application files for recovery purposes.

Enter information for the following fields:

  • Directory: Specify a directory that is writable by the same user as the Apache process.
  • Filename: Enter a file name for the Zip file including the .zip extension.

Click Confirm Settings to ensure that the backup operation can be performed.

Click Run Backup to create a .zip file of your Sugar application files.

Click Restore to restore the previous settings.

Click Cancel to exit the System Settings page without saving your changes.
