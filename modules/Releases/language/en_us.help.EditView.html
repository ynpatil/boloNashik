Releases Home

Use this page to create and maintain a list of your product releases.

To view the details of an existing release, click the name in the Release List. At the bottom of the page, the Release sub-panel displays the details.

To edit the details, revise the information in the Release form, and click Save.

To save your changes and create a new release, click Save&CreateNew.

To delete a release from the list, click the del icon corresponding to the name in the Release list; click OK to confirm the deletion.

To create a new release, click Create and enter the following information:

  • Release Version. Enter the release number for the new version.
  • Status. From the drop-down list, select Active to display the rate in the Release drop-down list; select Inactive if you do not want to display the release number.
  • Order. Enter a number to specify the order in which the number is displayed in the Release drop-down list.
