Forums

Use the Forums detail page to edit the forum and to create new threads for the topic.

To create a new thread, on the Threads page, click New Thread and enter the following information:

  • Title. Enter a title for the thread.
  • Sticky. Select this box if you want the thread to always be listed at the top of the threads list.
  • Body. Enter your comments in this field. To format the text with the HTML editor, click the down arrow and choose from the editor's menu options.

Click Save to create the thread; click Cancel to exit the page without creating the thread.
