Targets Home

The Targets Home page displays the following information:

  • A Search sub-panel where you can enter the target name to search for an existing target. To perform an advanced search using additional fields, click the Advanced Search tab. To customize and save the search layout and results, click the Saved Search & Layout tab.
  • A list of existing targets along with related information such as names and phone numbers.
  • To view target details, click the name or the View icon to the right of the target information in the list.
  • To edit the target details, click its name or the Edit icon to the right of the target information on the list.
  • To update or delete multiple target records, select them from the list, and use the Mass Update sub-panel.

A Shortcuts section that displays the following options:

  • Create Campaign. Click this option to create a new campaign.
  • Campaigns. Click this option to navigate back to the Campaings Home page from a campaign's detail page.
  • Create Target List. Click this option to create a new list of targets.
  • Target Lists. Click this option to navigate back to the Target Lists page from the detail page of a target list .
  • Create Target. Click this option to create a new target.
  • Targets. Click this option to navigate back to the Targets list from a target's detail page.
  • Import. Click this option to import a file containing target information from an external location.

A New Target quick form that you can use to create a target with only the required fields. You can enter additional details after you save the record.
