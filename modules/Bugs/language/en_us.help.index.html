Bug Tracker

Use the Bug Tracker module to create and manage bugs that are assigned to you or assigned by you to others.

The Bug Tracker Home page displays the following information:

  • A Search sub-panel where you can enter keywords such as the bug number and subject to search for specific bugs. To perform an advanced search using additional fields, click the Advanced Search tab. To customize and save the search layout and results, click the Saved Search & Layout tab. To restrain the search to bugs related to items that are assigned to you, select the Only my items box.

    A Shortcuts section that displays the following options:

    • Report Bug. Click this option to report a new bug. Alternatively, enter the required information in the New Bug quick form. You can specify additional information after you save it.
    • Bugs. Click this option to navigate back to the Bugs home page from a bug's detail page.
  • A New Bug quick form that you can use to create a bug with only the required fields. You can enter additional details after you save the new bug.
  • A list of existing bugs along with related information such as the status and type.
    • To view the details of a bug, click the name in the Bug list.
    • To export information on one or more bugs in .csv format to your local machine, select them in the list, click the Export link, choose Selected Records, and save the file. To export all the bugs listed on the page or the entire list, select Current Page and Entire List respectively.
    • To update or delete multiple bugs, select the bugs from the list, and use the Mass Update sub-panel.
