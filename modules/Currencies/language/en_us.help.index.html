Currencies

Use this page to define currencies and currency rates. Note that users can override the default currency that you specify on their My Account page.

To define a new currency, enter the following information:

  • Currency Name: Enter a name for the currency.
  • IS0 4217 Code: Enter the ISO number for the currency code.
  • Conversion Rate: Enter the conversion rate to the US dollar.
  • Currency Symbol:Enter the symbol for the currency.
