Leads

Use the Leads module to create and manage leads. Leads are early contacts in the sales process. After you have evaluated and assessed them, you can convert them into Contacts, Opportunities, and Accounts

The Leads Home page displays the following information:

  • A Search sub-panel where you can enter the name or the source to find a specific lead. To search specific modules, click the down-arrow and select the modules. To perform an advanced search using additional fields, click the Advanced Search tab. To customize and save the search layout and results, click the Saved Search & Layout tab.
  • A list of existing leads along with related information such as names and phone numbers.
    • To view the lead details, click the name or the View icon adjacent to the user name.
    • To edit the lead details, click the Edit icon adjacent to the user name; alternatively, click Edit on the detail page.
    • To update or delete multiple leads, select the leads from the list and use the Mass Update sub-panel.
  • A Shortcuts section that displays the following options:
    • Create Lead. Click this option to create a new lead. Alternatively, you can use the New Lead quick form to create the lead and add enter additional information after you save it.
    • Create From vCard. Click this option to import a vCard file from your local system. Browse to the location of the vCard, select it, and click Import vCard.
    • Leads. Click this option to navigate back to the Leads Home page from a lead's detail page.
    • Import. Click this option to import contact information from an external application or file.

    A New Lead quick form that you can use to create a lead with only the required fields. You can enter additional details after you create the lead.
