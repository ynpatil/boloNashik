Campaigns

Use the Campaigns module to create and manage campaigns for your organization.

The Campaigns Home page displays the following information:

  • A Search sub-panel where you can enter the campaign name to search for a specific campaign. To perform an advanced search using additional fields, click the Advanced Search tab. To customize and save the search layout and results, click the Saved Search & Layout tab.
    • Create Campaign. Click this option to create a new campaign. Alternatively, enter the required information in the New Campaign quick form. You can specify additional information after you save it.
    • Campaigns. Click this option to navigate back to the Campaigns Home page from a campaign's detail page.
    • Create Target List. Click this option to create a list of targets for a campaign.
    • Create Target. Click this option to create a campaign target.
    • Create Email Template. Click this option to create a template for a campaign email.
    • Email Templates. Click this option to view a list of existing campaign email templates.
    • Import Targets. Click this option to import target information from an external source.
  • A New Campaign quick form that you can use to enter the required information to create a new campaign. You can enter additional information after you create the campaign.
  • A list of existing campaigns along with related information such as names and phone numbers.
    • To view a campaign's details, click its name or the View icon to the right of the target information in the list
    • To export one or more campaigns, select them from the Campaign list, click the Export link, and choose Selected Records. To export all the campaigns listed on the page or the entire list, select Current Page and Entire List respectively.
    • To update or delete multiple campaigns, select them from the list, and use the Mass Update sub-panel.
