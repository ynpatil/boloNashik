Email Settings

Use this page to configure email settings for your organization. For email campaigns, you must configure tracker URLs and specify the number of emails sent out in a batch.

In the Email Notification Options sub-panel, enter information for the following fields:

  • From Name. Enter the name of the sender.
  • From Address. Enter the sender's address.
  • Notifications on? Select this option to send email notifications to users when a record is assigned to them.
  • Send Notifications by default for new users? Select this option to send an email notification to new users when you create their record in the User Management page.
  • Send Notifications from assigning user's email address? Select this option to send notifications from the email address of the user when responding to an assigned task.
  • Mail Transfer Agent. From the drop-down list, select the Mail Transfer Agent for your outbound emails. You can select either Sendmail or SMTP.

In the User Email Defaults sub-panel, specify the following information:

  • Compose email messages in this format. From the drop-down list, select either HTML format or Plain Text format.
  • Compose email using this client. From the drop-down list, select either the SugarCRM mail client, which is the default, or an external mail client such as Microsoft Outlook.

In the Outbound Email Options sub-panel, specify the following information:

  • Number of emails sent per batch. Enter the number of campaign emails you want to send out every batch.
  • Location of the campaign tracking files. Select either Default or User Defined. If you select Default, the system uses the value specified for the site_url parameter in the config.php file. If you select User Defined, you will need to specify the value in the field below.
To save your email settings, click Save; click Cancel to exit the page without saving your changes.
