Emails Group Inbox

Use the Group Inbox page to assign emails from the group inbox to a specific user or team. This page displays the following information:

  • A Search sub-panel where you can enter information such as the email status and subject to find a specific email.
  • An Assignment sub-panel that enables you to assign some or all group emails to a specific user or team using one of the following options:
    • Direct Assign: The user selects the users and assigns the emails to them.
    • Round Robin: The assigned emails are distributed equally among assigned users.
    • Least Busy: The system counts the emails marked as Assigned or Unread in the user inbox and routes assigned emails to the user who has the lowest number of assigned or unread emails. When all the users have an equal number of assigned emails, the system switches to the round robin mode.
  • A Group Inbox sub-panel where you can view existing group emails, check for new emails, and export some or all the emails to your local machine.
    • To check for new emails, click Check Mail button.
    • To export some emails, select them from the list, click Export, and choose Selected Records. To export all the emails on the page, choose Current Page; to export all the emails, select Entire list.
